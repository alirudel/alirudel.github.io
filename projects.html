  current projects

 

pacman

One of my very first Javascript exercises, this assignment was an exercise in understanding basic Javascript logic, loops and page width/boundaries. We were provided with four pacman images (but after my kid pointed out that Pacman doesn't have an eye, I updated the image to be a little more clean and classic).

eye exercise

The eye exercise was an opportunity to tie together HTML, CSS and Javascript into a single, simple project. The eyeballs track the mouse across the page using an "on mouse move" event, the eyes themselves are shapes created with CSS. I added a more exciting background to my version, photo by S O C I A L . C U T on Unsplash

orca sightings

When I first started learning about API's and using Mapbox I was delighted to find the whale sightings APIs from whalemuseum.org. This map project was an exercise in customizing the look of a map, and using a heat map to show sightings of a pod of whales in the Pacific Northwest.
